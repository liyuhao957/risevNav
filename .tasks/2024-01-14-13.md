## 部署文档
1. 初始部署步骤：
```bash
# 1. 在服务器上克隆代码
git clone [仓库地址]
cd [项目目录]

# 2. 安装依赖
npm install
pip install -r requirements.txt

# 3. 配置PM2
pm2 start backend/app.js --name "monitor-backend"

# 4. 配置Python脚本定时任务
crontab -e
# 添加：
# */5 * * * * cd /path/to/scripts && python3 huaweiSM.py
# */5 * * * * cd /path/to/scripts && python3 huaweiJZQ.py
```

2. 更新部署步骤：
```bash
# 1. 登录服务器
ssh user@server

# 2. 进入项目目录并更新代码
cd [项目目录]
git pull

# 3. 更新依赖（如果有新依赖）
npm install
pip install -r requirements.txt

# 4. 重启服务
pm2 restart monitor-backend
```

3. 常用维护命令：
```bash
# 查看Node.js服务状态
pm2 status
pm2 logs monitor-backend

# 查看Python脚本日志
tail -f /path/to/scripts/monitor.log

# 查看定时任务
crontab -l
```
